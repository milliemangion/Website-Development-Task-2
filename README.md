Pink Passport – Travel Blog Website
Project Overview
This project is a responsive travel blog website developed as part of a Website Development course. The website showcases travel destinations, provides an interactive quiz to help users discover their travel style, and demonstrates the use of modern front-end web development practices.
The site is built using HTML5, CSS3, Bootstrap, and JavaScript, with a strong focus on accessibility, responsiveness, and clean code structure. The project is version-controlled using GitHub and hosted via GitHub Pages.

Technologies Used
•	HTML5 – Semantic markup for page structure
•	CSS3 – Custom styling and layout enhancements
•	Bootstrap 5 – Responsive grid system, components, and utilities
•	JavaScript – Interactive quiz functionality
•	GitHub & GitHub Pages – Version control and hosting
 
Key Features
Responsive Design
•	Built using Bootstrap’s grid system
•	Fully responsive across desktop, tablet, and mobile devices
Destination Pages
•	Dedicated destination detail pages
•	Hero image carousel using Bootstrap
•	Embedded responsive Google Maps
•	Clear content sections for readability
Interactive Travel Quiz
•	10-question quiz built with JavaScript
•	Uses radio buttons and form validation
•	Displays results dynamically without page reload
Accessibility Considerations
•	Semantic HTML5 elements (header, nav, main, section, footer)
•	Alt text for all images
•	Clear heading hierarchy
 
JavaScript Functionality
The quiz functionality is implemented using JavaScript.
This approach keeps the logic simple, readable, and easy to maintain.
 
Design Decisions
•	Bootstrap was chosen to ensure responsiveness and consistent UI components
•	A hero carousel was used on destination pages to create visual impact
•	Content is broken into clear sections to improve readability and user experience
•	JavaScript was kept minimal to focus on clarity and functionality
 
Version Control
The project was developed with continuous commits to GitHub to: - Track progress over time - Provide regular backups - Demonstrate development
Each major feature was committed with descriptive commit messages.
 
Future Improvements
